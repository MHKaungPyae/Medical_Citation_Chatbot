# Auth UI Plan — Login + Register Pages

**Goal:** Add signup and login pages to the Medical Citation Chatbot frontend using Supabase Auth, styled with existing project design tokens.

**Status:** Planning complete, ready for implementation.

**Key files:**
- `frontend/app/login/page.tsx` — Login page
- `frontend/app/register/page.tsx` — Register page
- `frontend/components/AuthProvider.tsx` — Auth context provider
- `frontend/hooks/useAuth.ts` — Auth state hook
- `frontend/lib/supabase.ts` — Supabase client
- `frontend/lib/api.ts` — Auth-aware fetch wrapper

**Phases:** 4 phases, each independently verifiable.
